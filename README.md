Designed and developed a scalable, event-driven backend for a social media platform supporting real-time interactions and high user engagement

Implemented a hybrid API architecture combining RESTful endpoints with a GraphQL layer for flexible, efficient data querying and mutations across posts, comments, and user profiles

Built a comprehensive authentication and authorization system with JWT-based session management, role-based access control, and secure user registration/login flows

Developed full content management capabilities including CRUD operations for posts and comments, with reaction handling and custom GraphQL resolvers for optimized data fetching

Engineered a real-time chat system using Socket.io, supporting both private one-on-one messaging and group chat functionality with persistent message storage in MongoDB

Created modular data access layer with custom repository patterns and Mongoose schemas ensuring clean separation of concerns and maintainable code architecture

Integrated utility services including OTP generation, email notifications, password hashing, and token management for secure user workflows

Enforced robust input validation, error handling, and security middleware across both REST and GraphQL layers to ensure data integrity and protection

Leveraged TypeScript throughout the codebase for type safety, improved developer experience, and reduced runtime errors

Tech Stack: Node.JS, TypeScript, GraphQL, Socket.io, MongoDB, Mongoose, JWT, REST APIs, Socket.io, OTP/Email Utilities
